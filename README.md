Django Jewelry Shop App
Overview
The Django Jewelry Shop App is a web application designed for managing and showcasing a collection of exquisite jewelry items. It provides an intuitive and user-friendly platform for both customers and administrators to explore, purchase, and manage jewelry products.

Features
Product Catalog:
Display a stunning catalog of jewelry items with detailed descriptions, images, and pricing.
Categorize products for easy navigation (e.g., Rings, Necklaces, Earrings).

User Authentication:
Secure user authentication and authorization system.
Customers can register accounts, log in, and manage their profiles.

Shopping Cart:
Seamless shopping experience with an interactive shopping cart.
Add and remove items, update quantities, and proceed to checkout.

Order Management:
Track order history for customers.
Admins can manage orders, update order status, and view sales reports.

Admin Panel:
Easy-to-use admin interface for managing products, categories, and user accounts.
Add, edit, or remove products with detailed information.

Search Functionality:
Efficient search functionality for users to find specific products.
Responsive Design:

Mobile-friendly design for a seamless experience on various devices.
